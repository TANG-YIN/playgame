# 德州扑克游戏

这是一个支持2-6人玩家的德州扑克游戏实现。

## 功能特性

- **多玩家支持**: 支持2-6人同时游戏
- **完整游戏流程**: 翻牌前、翻牌、转牌、河牌、摊牌
- **牌力评估**: 自动识别所有牌型（高牌到同花顺）
- **投注系统**: 支持过牌、跟注、加注、全押、弃牌
- **盲注系统**: 自动收取小盲和大盲
- **筹码管理**: 完整的筹码追踪和分配

## 牌型等级

从高到低：
1. **同花顺** (Straight Flush)
2. **四条** (Four of a Kind)
3. **葫芦** (Full House)
4. **同花** (Flush)
5. **顺子** (Straight)
6. **三条** (Three of a Kind)
7. **两对** (Two Pair)
8. **一对** (One Pair)
9. **高牌** (High Card)

## 如何运行

### 环境要求

- Python 3.6+

### 运行游戏

```bash
python poker_game.py
```

### 使用说明

1. 程序启动后，输入玩家名称（至少2名玩家，最多6名）
2. 每位玩家初始筹码为1000
3. 游戏自动收取盲注（小盲10，大盲20）
4. 根据提示选择操作：
   - **1**: 过牌/跟注
   - **2**: 加注
   - **3**: 弃牌
   - **4**: 全押

## 游戏规则

### 牌堆
- 使用标准52张扑克牌（去掉大小王）
- 点数：2-10, J, Q, K, A
- 花色：♠, ♥, ♣, ♦

### 游戏流程

1. **翻牌前 (Preflop)**: 每位玩家发2张手牌
2. **翻牌 (Flop)**: 发3张公共牌
3. **转牌 (Turn)**: 发第4张公共牌
4. **河牌 (River)**: 发第5张公共牌
5. **摊牌 (Showdown)**: 剩余玩家比较牌力

### 下注规则

- **过牌**: 当不需要跟注时可以选择不增加下注
- **跟注**: 跟随当前最高注额
- **加注**: 增加注额（必须大于当前注额）
- **全押**: 押上所有筹码
- **弃牌**: 放弃当前回合

## 文件说明

- `poker_game.py`: 游戏主程序
- `poker_helper.py`: 德州扑克胜率计算器（原有功能）
